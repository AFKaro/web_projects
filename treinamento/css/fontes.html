Machine Learning

Um exemplo é o machine learning, que pode ser traduzido como aprendizado de máquina ou aprendizagem de máquina: esse é um conceito associado à inteligência artificial, razão pela qual é cada vez mais destacado pela mídia. Apesar disso, pouca gente compreende a ideia.
